Stream Cherry-Pick gives Administrators the ability to delete records from the Stream individually or in bulk.

== Description ==

**Note: This plugin requires [Stream](http://wordpress.org/plugins/stream/) to be installed and activated.**

If you're a control freak, this extension is for you!

When activated, this extension gives Administrators the ability to delete records from the Stream individually or in bulk.

Delete the past like it never happened!

**Development of this plugin is done [on GitHub](https://github.com/x-team/wp-stream-cherry-pick). Pull requests welcome. Please see [issues](https://github.com/x-team/wp-stream-cherry-pick/issues) reported there before going to the plugin forum.**
